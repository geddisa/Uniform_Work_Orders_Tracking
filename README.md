# Uniform Work Order Tracking App

This Streamlit application provides an interactive interface to track and update uniform work orders. It reads from and writes to the `Aramark (Vestis) Uniform Spreadsheet.xlsx` file.

## Prerequisites

- Python 3.8+
- The `Aramark (Vestis) Uniform Spreadsheet.xlsx` file must be in the root directory.

## Setup

1. Clone or download this repository.
2. Install the required dependencies:

   pip install -r requirements.txt
